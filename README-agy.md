# LTD AI 101

**Antigravity (agy) จากเปิดใช้ครั้งแรก จนใช้เป็นเครื่องมือทำงานจริงได้ ในประมาณ 60-90 นาที**

Version 0.3 | โดย Paint (ลงทุน Diary)

---

**BIG IDEA:** ถ้าคุณเข้าใจ 4 พื้นฐาน (AGY.md, slash command, skill, sub-agent) คุณสร้างเครื่องมือช่วยงานของตัวเองได้โดยไม่ต้องเขียน code

## ปัญหาที่คอร์สนี้แก้

ผมทำคลิปใช้ AI Coding Assistant มาก่อนแล้ว แต่อ่าน comment กลับมาเยอะมากว่า "ทำตามไม่ทัน" / "ลงเสร็จเปิดมาแล้วไม่รู้จะพิมพ์อะไร" / "งงต่อว่าจะทำอะไรต่อ / หรือผมมีคอร์สไหม?"

ผมเข้าใจ ผมเองก็เพิ่งหัด terminal ของจริงไม่กี่เดือน คอร์สนี้เลยเกิดจากความรู้สึกแบบนั้น คนที่เพิ่งจะเริ่มอยากได้คนเดินช้าๆ ไปกับคุณตั้งแต่ 0 ซึ่งที่ผมอยากสื่อคือ ไม่รู้อะไรก็แค่ถาม AI 

LTD AI 101 เลยเกิดขึ้นมา ไม่เหมือนกับคลิป YouTube เดิมของผม เพราะ AI น่าจะรู้ดีกว่าผมอีกว่ามันทำอะไรได้บ้างจริงไหม? เพียงคุณเปิด folder นี้ใน Antigravity (ที่ลงไว้ในเครื่องอยู่แล้ว) แล้ว agy เองจะเป็นคนสอนคุณตามบท ทีละ step หยุดรอคุณตอบ หยุดรอคุณกด แล้วค่อยไปต่อเมื่อคุณพร้อม ค่อยๆ ทำไปกับคุณ

---

## สิ่งที่คุณจะได้หลังจบคอร์สนี้

คอร์สนี้สร้างของชิ้นเดียวกันแต่จะค่อยๆ เพิ่มความยากใน 6 บท เริ่มจากของง่ายๆ แล้วค่อยๆ ปรับขึ้นจนกลายเป็นเครื่องมือที่เอาไปปรับใช้งานได้จริง:

- ของชิ้นนั้นคือ **`/brief TICKER`** เครื่องมือสรุปหุ้น 1 ตัว (ทำอะไร, ข่าว 7 วันล่าสุด, bull/bear, ถามอะไรก่อนซื้อ) ที่ save ไฟล์ markdown ลง folder `briefs/` ของคุณเอง
- L1 ทำเวอร์ชั่น 0 เป็น slash command ขั้นต้น (คำสั่งลัดที่ขึ้นต้นด้วย `/`)
- L2 ยกระดับเป็น **skill** (ทักษะที่ agy เรียกใช้ซ้ำได้) แล้วใส่ "เสียงนักลงทุน" ของคุณเองเข้า `AGY.md` ของโปรเจค
- L3 ใส่ earnings transcript เข้ามาเป็น source ที่ agy ใช้เขียน brief จริง (ไม่ปล่อยให้ agy เดาจาก training data เก่า) + วิธีคุม cost (เลือก model ให้คุ้ม, การดู token และการจัดการ context)
- L4 แตก research ออกเป็น 3 sub-agent ทำงานขนานกัน (fundamentals อ่าน 10-K, earnings อ่าน transcript, news & sentiment ใช้ websearch)
- L5 deploy showcase ขึ้น Vercel
- L6 ยกระดับด้วยระบบ **Memory (MEMORY.md)** บันทึกความทรงจำระยะยาวข้ามวัน พร้อมพนักงานย่อยคนที่ 4 สายขวางโลกอย่าง **"ผู้ขัดคอสะกิดเตือน (Devil's Advocate)"** ที่ช่วยวิเคราะห์ข้อเสียและคัดค้านเพื่อลบอคติการลงทุน

ของจริงทั้งหมด ไม่ใช่ demo จบคอร์สคุณจะมี `/brief` ใช้ได้ในเครื่องของคุณเอง พร้อมไฟล์ briefs/ ที่ generate จากทุกตัวที่คุณ run และ URL showcase ที่ share ให้คนอื่นดูได้

ของพวกนี้มีไว้เพราะ AI ทำให้คนที่ไม่ใช่ดเวลอปเปอร์ มีเวลา build conviction ได้เร็วพอที่จะเป็นนักลงทุนที่เลือกหุ้นเอง ไม่ใช่แค่ DCA ตาม ETF อย่างเดียว

## คอร์สนี้สำหรับใคร

- คนที่ดาวน์โหลด Antigravity มาแล้ว แต่ยังไม่ได้ลองทำอะไรจริง
- คนที่เคยใช้ ChatGPT หรือ Gemini Web มาก่อน แต่ยังไม่เคยให้ AI เห็นไฟล์ในเครื่องตัวเอง
- คนที่กลัว terminal (หน้าจอดำพิมพ์คำสั่ง) แต่อยากใช้ AI ทำงานได้มากกว่าแชทตอบเฉยๆ
- นักลงทุน/นักทำคอนเทนต์ ที่อยากเริ่มสร้างระบบช่วยตัวเอง ไม่ใช่ใช้แอพคนอื่นต่อไปเรื่อยๆ

**ไม่ต้องเขียน code เป็นมาก่อน.** agy เป็นคนทำงานหนักให้ คุณแค่พิมพ์ภาษาคน

## โครงสร้างคอร์ส

6 บทเรียน รวมประมาณ 60-90 นาที ทุกบทต่อยอด **ของชิ้นเดียวกัน** คือ `/brief TICKER`.

| Lesson | สิ่งที่จะสร้าง / เพิ่มเข้า `/brief`                                            | เวลา      |
| ------ | ------------------------------------------------------------------------------ | --------- |
| 1      | โปรเจคแรก + `AGY.md` + `/brief` v0 (slash command, prompt ตรงๆ บันทึก briefs/) | ~20-25 min |
| 2      | ยก `/brief` เป็น skill (ทักษะ reusable) + ใส่เสียงนักลงทุนของคุณเข้า `AGY.md` | ~15-20 min |
| 3      | ใส่ earnings transcript เป็น source + คุม cost (model picker, token management)  | ~25 min    |
| 4      | แตก `/brief` เป็น 3 sub-agent ขนาน (fundamentals/10-K, earnings/transcript, news+sentiment/websearch) | ~15 min    |
| 5      | Deploy showcase ขึ้น Vercel                                                    | ~15-20 min |
| 6      | พนักงานจับผิด (Devil's Advocate) + ความจำข้ามเซสชัน (Long-term Memory)          | ~30-40 min |

แต่ละบทต่อจากของบทก่อน Lesson 2 ยกระดับ `/brief` ที่ Lesson 1 สร้าง Lesson 4 แตก skill ที่ Lesson 2 ทำ ข้ามไปไม่ได้

## วิธีเริ่ม

คอร์สนี้ไม่ได้ออกแบบให้นั่งอ่าน คุณกำลังเปิด folder นี้ in Antigravity อยู่แล้ว agy จะเป็นคนสอนคุณ

### ยืนยันก่อนว่าคุณพร้อม

มอง chat ของ Antigravity ตอนนี้ ถ้าเห็น chat input ที่พิมพ์ได้ และดู file tree ทางซ้ายเห็น folder `ltd-ai-101` (มี `README-agy.md`, `AGY.md`, `lesson-modules-agy/`) แสดงว่าพร้อม

ถ้ายังมียังไม่เห็นแบบนั้น ตรวจ: คุณเปิด folder `ltd-ai-101` ใน Antigravity แล้วใช่ไหม (ไม่ใช่ folder อื่น)

### พิมพ์คำนี้ใน chat

```

```

จบ lesson 1 แล้วค่อยพิมพ์ `Start Lesson 2`, `Start Lesson 3`, ตามลำดับ

## วิธีทำงานของแต่ละ lesson

ทุก lesson หน้าตาเหมือนกัน:

1. agy อ่านไฟล์ lesson แล้วเริ่มคุยกับคุณ in chat
2. คุณทำตาม paste โค้ดบล็อก ยืนยันสิ่งที่เห็นบนจอ
3. agy หยุดที่ทุก checkpoint ที่สำคัญ เพื่อเช็คว่าของที่เพิ่งทำมัน work จริง
4. จบ lesson agy จะบอกว่าให้พิมพ์อะไรเพื่อขึ้นบทถัดไป

คุณไม่ได้กำลังนั่งอ่าน documentation. คุณกำลังคุยกับ agy แบบมี guide ขณะที่สร้างของไปด้วย

**สำคัญ: คอร์สนี้ใช้ 2 หน้าต่าง.** ตั้งแต่ Lesson 1 คุณจะเปิด Antigravity หน้าต่างที่ 2 ที่ folder `my-first-project`. หน้าต่าง 1 (folder `ltd-ai-101` หน้าต่างนี้) คือผู้สอน หน้าต่าง 2 คือ workshop ของคุณ paste prompt ที่ผู้สอนส่งให้ และดูผลลัพธ์ที่ agy สร้าง คุณสลับ 2 หน้าต่างไปมาตลอดคอร์ส (Cmd+Tab / Alt+Tab). อย่าปิดหน้าต่าง 1 ระหว่างคอร์ส

## กฎเดียวของคอร์สนี้

ทำตามลำดับ อย่าข้าม

ทุก lesson ต่อยอด `/brief` ตัวเดิมจากบทก่อน Lesson 2 ใช้ slash command จาก Lesson 1. Lesson 4 แตก skill จาก Lesson 2. Lesson 5 deploy ของที่สร้างมาทั้งหมด ถ้าข้าม คุณจะติดและไม่รู้ว่าติดที่ไหน

ถ้าระหว่างทางมี step ไหนที่ทำตามไม่ทัน หยุดบอก agy ว่า "ช้าหน่อย" หรือ "ไม่เข้าใจ ขอใหม่" agy จะกลับมาอธิบายให้ ไม่ต้องเร่ง
